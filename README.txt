Simple Instagram image feed fetchers by hashtags.

== Description ==

It fetches list of images tagged by given hashtags in the shortcodes. You may include shortcode in post to render the list of images organized in masonry layout with click to enlarge view feature.


== Installation ==

1. Upload the project to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Save your settings then place shortcode `instalivit` eg. <br> [instalivit hashtag='tag1,tag2,tag3'] in posts
